# Classic — Amrutham Authentics

Traditional **Gold + Dark Brown** palette. The most heritage-forward of the
three variants — a warm, restrained, temple-cuisine feel.

## Palette
| Token | Hex | Role |
|---|---|---|
| `--brand-gold` | `#b8873e` | Primary warm — CTAs, highlights |
| `--brand-gold-deep` | `#9e6f28` | Gold accents, deeper hover state |
| `--brand-brown` | `#4a2c1a` | Primary dark — headlines, footer, nav active |
| `--brand-brown-deep` | `#2e1a0e` | Deepest ink — body text |
| `--brand-terracotta` | `#c55a3c` | Accent — menu prices, featured badges |
| `--brand-olive` | `#6b7535` | Accent — dietary/veg tags |
| `--cream` | `#fdf8f0` | Background |

## Running
```bash
npm install
npm run build
npm run deploy   # deploys to Cloudflare Pages as "amrutham-classic"
```
